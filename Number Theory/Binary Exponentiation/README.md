This is an algorithm to find the value of <code>BASE<sup> (POWER)</sup></code>

When `POWER` is ODD

`
  RESULT = RESULT * BASE
  POWER = POWER - 1
`

When `POWER` is EVEN

`
  BASE = BASE * BASE
  POWER = POWER / 2
`
